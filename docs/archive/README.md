# Archive — Original Design Documents

This folder contains the original brainstorming, research, and early design documents that were used to develop the canonical spec for **Skully Bones Treasure Adventure**. They are preserved for historical reference.

**The authoritative specification is the root [README.md](../../README.md).** If anything in these archived documents contradicts the root README, the README takes precedence.

## Contents

| Document | Description |
|---|---|
| README_old.md | Previous version of the root README before consolidation. |
| TheGame.md | Early core game concept and v1 scope draft. |
| Star Battle.md | Rules reference for the 1-star Star Battle model. |
| The classic N-Queens puzzle.md | Theme and historical context for the queen aesthetic. Not the canonical rule set. |
| Reward System.md | Early reward model notes (stars, coins, later-phase ideas). |
| reward and progression system.md | Early progression loop and deferred system notes. |
| Questions and Answers.md | **Raw research and brainstorming.** Contains competitor analysis, unedited Q&A notes, and references to features from other shipped games. Some terminology and feature assumptions in this document do not reflect project decisions. |

## Important Notes

- These documents use inconsistent terminology (`queen`, `piece`, `star`). The canonical term is **token**.
- `Questions and Answers.md` includes competitor research (e.g., Kwalee games, Reddit threads) that was used to inform decisions but does not represent project specs.
- Reward and progression content from multiple documents has been consolidated into the root README.
- Features discussed in these documents (pets, boosters, timed levels, leaderboards, events) are deferred and not part of V1 scope.
